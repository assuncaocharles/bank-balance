# Bank Balance Client

## Run 

* First of all, there's an production version in the api. It my be of your interest to access it [cliking here](http://localhost:3000/index.html). 

If you want to run this development version, first install the dependencies running `npm i` on this project root and the run `npm start`. The project will run on http://localhost:3002 ( Make sure you have the API running on http://localhost:3000 )

#### P.S: THIS PROJECT IS MERELY TO INTERACT WITH THE API, NO INTENTION TO SERVER AS PRODUCTION FRONT-END
